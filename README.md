# hubot-aws-ec2

Adds AWS EC2 to Hubot

See [`src/scripts/aws.js`](src/scripts/aws.js) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-aws-ec2 --save`

Then add **hubot-aws-ec2** to your `external-scripts.json`:

```json
[
  "hubot-aws-ec2"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
